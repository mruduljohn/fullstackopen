Here is a simple flow chart:

```mermaid
sequenceDiagram
    participant User
    participant Browser
    participant Server
    User->>+Browser: Enters text and clicks submit
    Browser->>+Server: POST request to /new_note endpoint
    Server->>+Database: Saves new note to database
    Database-->>-Server: Returns success message
    Server-->>-Browser: Returns success message
    Browser-->>-User: Shows success message and updated note list
```
